# 🌟 星座探索 — 儿童星座科普互动网站

为 **小朋友**打造的星空探索之旅！通过连线游戏认识 88 个星座，每完成一个连线就能看到可爱的卡通星座形象从星空中浮现。

## ✨ 特色功能

- **🎮 连线游戏** — 拖拽手指把星星连起来，完成星座图案
- **🎨 卡通形象叠加** — 连完所有星后，精美的卡通形象会从画面中淡入
- **📖 星座百科** — 每个星座都有神话故事 + 趣味知识
- **📊 星座图鉴** — 浏览已解锁的星座，追踪学习进度
- **👶 儿童友好** — Claymorphism 软萌风格，Fredoka + Nunito 圆体字，7 岁孩子轻松上手

## 🛠 技术栈

| 分类 | 技术 |
|------|------|
| 页面 | HTML5 / CSS3 |
| 逻辑 | 原生 JavaScript（无框架） |
| 图形 | Canvas 2D（星图绘制 + 连线交互） |
| 图片 | SVG（30 张星座卡通形象） |
| 存储 | localStorage（游戏进度持久化） |
| 部署 | 纯静态网站，任意托管即可 |

## 🚀 快速开始

1. 克隆仓库：
   ```bash
   git clone https://github.com/daka-agent/Constellation.git
   ```

2. 打开 `index.html` 即可运行——**零依赖，无需构建**

3. 或部署到任意静态托管：
   - **GitHub Pages**：Settings → Pages → 选 `main` 分支
   - **Vercel / Netlify**：直接导入仓库即可
