# 🌟 星座探索 — 儿童星座科普互动网站

为 **小朋友** 打造的星空探索之旅！通过连线游戏认识 88 个星座，每完成一个连线就能看到可爱的卡通星座形象从星空中浮现。

---

## 🎮 在线玩

无需安装，点击即玩 👉 **[https://daka-agent.github.io/Constellation/](https://daka-agent.github.io/Constellation/)**

> 也可安装为桌面/手机 App（支持 PWA 离线使用）。

---

## ✨ 功能一览

### 🎮 核心玩法
- **连线游戏** — 拖拽手指把星星连起来，完成星座图案
- **难度递增** — ★入门 → ★★进阶 → ★★★挑战 → ★★★★大师，逐级解锁
- **卡通形象叠加** — 连完所有星后，精美卡通星座形象淡入登场
- **通关分析图** — 五边形雷达图 + S/A/B/C/D 评级 + 历史最佳对比

### 📖 学习辅助
- **星座百科** — 88 篇神话故事 + 趣味知识
- **🔊 TTS 朗读** — 浏览器语音合成，自动朗读神话故事（中文）
- **星图全景** — 3000×3000 圆形星图，88 星座按季节 6 色分组，可缩放漫游
- **搜索筛选** — 按难度 / 季节 / 状态 / 排序多维度筛选，名称拼音搜索

### 🏆 激励系统
- **成就徽章** — 10 种徽章（连线 / 故事 / 连击 / 探索 4 大类）
- **新手引导** — 3 步交互式教程，初次访问自动触发
- **图鉴解锁** — 浏览已解锁星座，追踪学习进度

### 📱 技术体验
- **PWA 离线支持** — 可安装到桌面/手机，离线仍可访问
- **音效系统** — 10 种 Web Audio API 合成音效，零外部文件
- **触摸优化** — 移动端手势流畅，60fps Canvas 渲染
- **👶 儿童友好** — Claymorphism 软萌风格，Fredoka + Nunito 圆体字

---

## 🛠 技术栈

| 分类 | 技术 |
|------|------|
| 页面 | HTML5 / CSS3 |
| 逻辑 | 原生 JavaScript（零框架依赖） |
| 图形 | Canvas 2D（星图绘制 + 连线交互 + 全景漫游） |
| 图片 | SVG × 88（星座卡通形象） |
| 音效 | Web Audio API（OscillatorNode 合成） |
| 语音 | Web Speech API（TTS 朗读） |
| 存储 | localStorage（进度 / 徽章 / 历史） |
| 离线 | Service Worker（Cache First 策略） |
| 部署 | 纯静态网站，任意托管即可 |

## 📁 项目结构

```
xingzhuo/
├── index.html              # 入口页面（PWA meta + SW 注册）
├── manifest.json           # PWA 应用清单
├── service-worker.js       # Service Worker（离线缓存 v2.0.0）
├── css/
│   └── style.css           # 设计系统 + 响应式 + 动画
├── js/
│   ├── data.js             # 88星座数据（坐标 / 神话 / 知识 / 难度）
│   └── game.js             # 状态管理 / 页面路由 / Canvas渲染 / 游戏逻辑
├── imgs/
│   ├── *.svg               # 88 张儿童友好卡通星座形象
│   └── icons/              # 8 种尺寸 PNG 图标 (72×72 ~ 512×512) + favicon
└── README.md
```

## 🚀 快速开始

1. 克隆仓库：
   ```bash
   git clone https://github.com/daka-agent/Constellation.git
   ```

2. 打开 `index.html` 即可运行 —— **零依赖，无需构建**

3. 或部署到任意静态托管：
   - **GitHub Pages**：Settings → Pages → 选 `main` 分支
   - **Vercel / Netlify**：直接导入仓库即可
   - **CloudStudio**：上传目录即部署
